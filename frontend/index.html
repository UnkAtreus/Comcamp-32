WHAT IS COMCAMP คอมแคมป์คืออะไร
โครงการฝึกอบรมเชิงปฏิบัติการคอมพิวเตอร์เบื้องต้น ครั้งที่ 32 น้อง ๆ เคยสงสัยกันบ้างหรือเปล่า ว่าวิศวะคอมทำอะไรกัน? สนุกไหม? แล้วเสน่ห์ของวิศวะคอมคืออะไรกันนะ? งั้นมาเข้าค่าย Comcamp 32nd กันเถอะ! มาลองสัมผัสความเป็นนักศึกษาวิศวะคอมพิวเตอร์ มาเรียนรู้ มาลงมือทำจริง ซึมซับประสบการณ์ ใหม่ ๆ และมาพับกบ เอ้ย! พบกับเพื่อน ๆ มากหน้าหลายตา และที่สำคัญค่ายนี้... ฟรีทั้งค่าย พี่ ๆ กำลังรอน้อง ๆ มาเป็นส่วนหนึ่งของค่ายอยู่นะ รีบสมัครกันเข้ามา เยอะ ๆ น้า

Learning

Pre programming

เตรียมพร้อมสำหรับการผจญภัย ผ่านการเรียนรู้และฝึกกระบวนการคิดโดยใช้ผังงาน (Flowchart) รวมทั้งการแปลงความหมายออกมาเป็นรหัสเทียม (Pseudocode) เพื่อให้คำสั่งจากภารกิจยาก ๆ สามารถเข้าใจได้ง่ายยิ่งขึ้น!

C programming

ร่ำเรียนเคล็ดวิชาลับภาษาซี ตั้งแต่ขั้นพื้นฐานจนถึงขั้นประยุกต์ เพื่อให้เหล่าผู้กล้าสามารถบรรลุภารกิจพื้นฐาน และใช้ศึกษากระบวนท่าภาษาอื่นต่อได้ พร้อมทั้งมีโจทย์สุดท้าทายที่รอทดสอบความสามารถของผู้กล้าทุกท่านอยู่!

Microcontroller board

สัมผัสเครื่องมือสุดล้ำค่าอย่าง Microcontroller board และร่วมศึกษาทักษะสำคัญในการควบคุมบอร์ดผ่าน Arduino IDE ตั้งแต่พื้นฐานไปจนถึงขั้นประยุกต์จนเป็นเครื่องมือที่น่าอัศจรรย์นามว่า Internet of things (IOT)

Machine Learning

เหนือกว่าด้วยความสามารถสุดพิเศษที่มนุษย์คนใดไม่อาจเทียบได้ ทำความเข้าใจศาสตร์แห่ง machine learning ด้วยภาษา Python ขั้นต้นและลงมือสร้าง machine learning ขึ้นมาเป็นของตนเอง!

Brainstorming Techniques

ฝึกกลยุทธ์ให้เชี่ยวชาญและมีประสิทธิภาพสูงสุด ผ่านภารกิจสุดสนุกแต่แฝงไปด้วยหลักการและเทคนิคอันน่าอัศจรรย์ เพื่อดึงศักยภาพของเหล่าผู้กล้าออกมาอย่างสร้างสรรค์และติดตัวไปใช้ได้แม้อยู่ภายนอกเกม!

Faq

ค่ายจัดขึ้นที่ไหนคะ?

ค่าย Comcamp ครั้งที่ 32 ของเรา จัดขึ้นที่ภาควิชาวิศวกรรมคอมพิวเตอร์ มหาวิทยาลัยเทคโนโลยีพระจอมเกล้าธนบุรีค่ะ

ค่ายจัดวันไหนครับ?

ค่าย Comcamp ครั้งที่ 32 ของเรา จัดขึ้นในระหว่างวันที่ 11 - 15 เมษายน พ.ศ.2563 ครับ

รับจำนวนกี่คนคะ?

ค่าย Comcamp ครั้งที่ 32 ของเรา รับนักเรียนเข้าร่วมโครงการจำนวน 100 คนค่ะ ซึ่งทางเราจะทำการคัดเลือกน้อง ๆ จากคำถามรับสมัคร และจะติดต่อกลับเมื่อน้อง ๆ ได้รับคัดเลือกเข้าร่วมโครงการค่ะ

รับนักเรียนชั้นไหนบ้าง และถ้าผมอยู่ ม.3 ขึ้น ม.4 สามารถเข้าร่วมได้ไหมครับ?

ค่าย Comcamp ครั้งที่ 32 ของเรา รับนักเรียนระดับชั้นมัธยมศึกษาปีที่ 4 - 6 ปีการศึกษา 2562 และน้อง ๆ ปวช. เตรียมวิศวกรรมศาสตร์ เท่านั้นครับ สำหรับน้อง ๆ ม.3 ขึ้น ม.4 ขอให้อดใจรอมาพบกันใหม่ในปีหน้านะครับ

เสียค่าใช้จ่ายกี่บาท และต้องจ่ายค่าอาหารเองไหมคะ?

ค่าย Comcamp ครั้งที่ 32 ของเราเป็นค่ายฟรีค่ะ ดังนั้นจึงไม่มีการเก็บค่าอาหาร รวมถึงค่าที่พัก แต่จะมีค่ายืนยันสิทธิก่อนเข้าค่ายจำนวน 200 บาท ซึ่งทางค่ายจะคืนเงินจำนวนดังกล่าวให้เมื่อน้องเข้าร่วมครบเวลากิจกรรมค่ะ

จำเป็นต้องเข้าร่วมกิจกรรมทุกวันหรือเปล่าครับ และทางค่ายมีที่พักเตรียมไว้หรือเปล่าครับ?

เพื่อความต่อเนื่องในการทำกิจกรรม พี่ขอให้น้องๆมาเข้าร่วมทุกวันนะครับ สำหรับที่พัก พี่ๆได้จัดเตรียมหอพักนักศึกษาไว้ให้กับน้องๆแล้วครับ ซึ่งมีการรักษาความปลอดภัยตลอด 24 ชั่วโมงครับ

ถ้าได้เข้าค่ายจะมีโควต้าพิเศษเข้ามหาลัยไหมคะ?

ทางภาควิชาวิศวกรรมคอมพิวเตอร์ ไม่มีโควต้ารับนักศึกษาเข้าผ่านโครงการค่าย Comcamp แต่น้องๆสามารถนำภาพกิจกรรมภายในค่ายและประกาศนียบัตร ไปใช้ใน Portfolio เพื่อนำไปสมัครเข้าภาควิชาวิศวกรรมคอมพิวเตอร์ได้ครับ

สามารถสมัครค่ายได้อย่างไรครับ?

น้อง ๆ ที่สนใจเข้าร่วมค่ายสามารถสมัครได้ผ่านทางเว็บไซต์ ตั้งแต่วันที่ 10 กุมภาพันธ์ 2563 ถึงวันที่ 9 มีนาคม 2563 ครับ
